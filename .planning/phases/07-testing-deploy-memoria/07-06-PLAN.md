  <name>Task 2: Compilar PDF de la memoria (instrucciones + intento)</name>
  <files>memoriaTFG/Plantilla TfG/proyect.tex</files>
  <action>
    Intentar compilar el PDF localmente. Si pdflatex no está instalado en el sistema,
    documentar el proceso alternativo con Overleaf.

    **Intento de compilación local:**
    ```bash
    cd "memoriaTFG/Plantilla TfG"

    # Compilación completa (4 pasos para referencias cruzadas y bibliografía)
    pdflatex proyect.tex
    bibtex proyect
    pdflatex proyect.tex
    pdflatex proyect.tex

    # Si la compilación tiene éxito:
    cp proyect.pdf ../../memoriaTFG/bargain-memoria.pdf
    ```

    **Si pdflatex falla o no está instalado:**
    1. Subir los archivos de `memoriaTFG/Plantilla TfG/` a Overleaf.com
    2. En Overleaf: New Project → Upload → seleccionar todos los archivos del directorio
    3. Compilar desde la interfaz web
    4. Descargar el PDF resultante
    5. Guardar como `memoriaTFG/bargain-memoria.pdf`

    **Errores comunes y soluciones:**
    - `! LaTeX Error: File 'pclass.cls' not found`: verificar que pclass.cls está en el mismo
      directorio que proyect.tex
    - `Package fontenc Error: Encoding file 'helvet.fd' not found`: instalar paquete `tex-helvetic`
      en TeX Live: `tlmgr install helvetic`
    - `Undefined control sequence \hacerportada`: pclass.cls no está cargado correctamente —
      verificar la línea `\documentclass{pclass}`
    - Errores de bibliografía: asegurarse de que `pfcbib.bib` existe en el directorio

    Crear `memoriaTFG/BUILD-INSTRUCTIONS.md` con estas instrucciones si no existe.
  </action>
