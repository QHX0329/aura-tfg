         - Spawns `multiprocessing.Process(target=_run_spider_process, args=(spider_path,))`
         - `p.start()`, `p.join(timeout=3600)` (1h max)
         - If `p.exitcode != 0`: raise `self.retry(exc=RuntimeError(...))`
         - Return `{"status": "ok", "spider": spider_name}`
