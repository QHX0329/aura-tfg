Price.objects.filter(product=product, verified_at__gte=cutoff)
    .annotate(day=TruncDay("verified_at"))
    .values("day", "store__id", "store__name")
    .annotate(min_price=Min("price"), max_price=Max("price"), avg_price=Avg("price"))
    .order_by("day")

Stale price rule (locked decision):
- scraped prices: stale after 48h (verified_at < now - 48h)
- crowdsourced prices: stale after 24h
- NEVER hard-delete prices; only set is_stale=True
- Price history purge: DELETE records older than 90 days (this IS a hard delete — for cleanup only)
